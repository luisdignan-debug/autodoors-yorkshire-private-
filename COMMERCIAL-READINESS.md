# Commercial Readiness Notes

## Positioning

This is a done-for-you trade operating dashboard, not generic CRM and not accounting software.

Primary workflow:

lead -> quote -> deposit -> supplier order -> delivery -> installation -> invoice/balance -> review -> closed.

## Suggested Packages

- Starter: setup, email sync, leads, Today dashboard, exports.
- Pro: Starter plus invoices, supplier invoices, finance view, support.
- Operations: Pro plus technician schedule, weekly review, workflow tuning.
- Custom: multi-user, custom templates, deeper integrations.

## Setup Checklist For Each Client

- confirm trade type and workflow;
- configure business details;
- confirm VAT mode;
- add bank details;
- configure email sync;
- set admin login;
- add technician;
- run one live test lead;
- generate one invoice PDF;
- export data;
- explain accounting boundaries.

## Support Boundaries

Do not promise statutory accounts, VAT filing, MTD, payroll, CIS, legal compliance, guaranteed lead conversion or bank reconciliation.

## Monthly Maintenance

- check backups/exports;
- check failed email sync;
- review stuck jobs;
- review money to collect;
- review supplier owed;
- check invoice numbering;
- review feature flags remain safe.

## Sales Demo

Use `/demo` or `DEMO_MODE=true` for sales calls. Demo data is fake and should never be mixed with live customer data.
