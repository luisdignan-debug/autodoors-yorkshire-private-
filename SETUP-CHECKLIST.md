# Client Setup Checklist

Use this before handing the dashboard to a client.

- Business details entered.
- Invoice settings complete.
- VAT setting confirmed.
- Bank details confirmed.
- Admin username and password set.
- Email sync tested.
- First lead added.
- First quote recorded.
- First invoice PDF generated.
- First technician or work order added if the business installs on site.
- Data export downloaded.
- Client shown Today page on mobile.
- Client told this is not accounting software.

Ready means the client can open **Today** and understand what to do next.
