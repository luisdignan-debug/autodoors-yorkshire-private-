# Auto Doors Yorkshire Operating Dashboard

This dashboard is an operational control layer for a trade business.

It helps you see:

- new leads that need a reply;
- quotes that need chasing;
- deposits and balances due;
- supplier orders that need placing or checking;
- installs that need booking;
- supplier invoices owed;
- professional customer invoices;
- exports of your data.

It is not accounting software. Keep using your accountant, Xero, QuickBooks, bank app, payroll, VAT/MTD tools and bookkeeping process.

## Daily Use

1. Open **Today**.
2. Work from the top down.
3. Open each job and press the next safe action.
4. Check **Finance** for money to collect and supplier invoices owed.
5. Export data regularly from **System**.

## Safe Defaults

Emails, SMS, WhatsApp and calendar sync do not send automatically by default.

The app creates previews and draft text first. Sending only works when the relevant feature flag and provider settings have been deliberately enabled.

## Your Data

You can export:

- leads;
- jobs;
- customer invoices;
- customer payments;
- supplier invoices;
- supplier payments;
- supplier emails;
- all data as JSON;
- the tracker workbook.

Use **System -> Download my data**.

## If Email Sync Fails

1. Check the mailbox password has not changed.
2. Check IMAP/SMTP host and username.
3. Run sync again from the dashboard.
4. If it still fails, export your data before changing settings.

## Accountant Note

Use dashboard figures as operational estimates only. Your accountant/bookkeeper should confirm formal accounts, VAT, CIS, payroll and statutory filings.
